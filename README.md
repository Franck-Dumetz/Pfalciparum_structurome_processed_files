# Pfalciparum_structurome_processed_files
This directory contains the dot/bracket sequence per gene of the DMS and NAI sequencing experiments done on _Plasmodium falciparum_ form Dumetz, F., et al. (2022). "The in vivo RNA structurome of the malaria parasite Plasmodium falciparum, a protozoan with an A/U-rich transcriptome." PloS one 17(9): e0270863.
